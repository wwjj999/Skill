# 通义灵码项目专属规则 (Agents-MD Pro 协议)

## 核心原则

您属于 Agents-MD Pro 工程架构中的智能助手，您的每一项编码、问答和代码段推荐均需遵循工程的核心边界约束协议。

1. **协议入口**：在帮助研发或重构前，请主动检索并阅读项目根目录的 `AGENTS.md`。
2. **本地增强**：了解 `.agents/skills` 下配置了哪些能力，尽量采用已有本地能力来减少您的计算开销。
3. **工作流**：必须遵守 `PROJECT_GOVERNANCE.md` 中制定的 5 步研发规范（含改动前的迷你设计要求）。

违反 `AGENTS.md` 约束（如擅自脱离项目规范的单体文件更改，无架构支撑的设计，不打结构化日志）是不被允许的。永远以可控、低风险、架构严谨的姿态生成所有响应。
