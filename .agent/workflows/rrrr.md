# /rrrr - Refresh Context

This command refreshes the AI's understanding of project governance.

## Steps

1. Read `AGENTS.md` completely
2. Read `PROJECT_GOVERNANCE.md` completely
3. Read `PROJECT_STATUS.md` completely
4. Confirm understanding with a brief summary

## Usage

Type `/rrrr` when:

- Starting a new session
- AI seems to "forget" project rules
- After a long conversation
- Before making important changes

## Expected Output

AI should confirm:

- ✅ AGENTS.md loaded
- ✅ PROJECT_GOVERNANCE.md loaded
- ✅ PROJECT_STATUS.md loaded
- Brief summary of current project state
